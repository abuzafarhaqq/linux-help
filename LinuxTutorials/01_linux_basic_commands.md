### Remove:
---

```
rm file1 file2
rm *.txt
rm -rf ~/Document/test/text/
```

- removes file1 and file2
- removes all file with extension txt
- removes all files under `~/Documents/test/text/` folder.

### Copy:
---

```
cp dir1 dir2
```

### Move:
---

```
mv dir1 dir2
```

> Move command also used for `RENAME` file.
