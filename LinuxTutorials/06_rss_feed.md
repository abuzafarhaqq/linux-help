# RSS Feed Reader

---

Using RSS, you can get many website's post for offline, also you can use it for downloading the podcast.

### Install RSS Feed Reader

Ubuntu: sudo apt install newsboat

Termux: pkg install newsboat

### Configure

1. Create folder named `.newsboat` using: `mkdir ~/.newsboat`

2. Create a file named `urls` using: `touch ~/.newsboat/urls`

3. And now paste this urls into `urls`:
